Lights, Camera, Action!

Your Montage application is running fine. You're looking at the content of your main component located at ui/main.reel. This modular component encapsulates the html, css, javascript, and any other bundled resources it needs to do its thing. Feel free to poke around inside.

Minit provides several commands to help you build your app; run minit commands from inside your application directory.

Add a component

  1. minit create:component -n greeting
  2. Add the greeting to your main component's declaration "greeting": { "prototype": "ui/greeting.reel", "properties": { "element": {"#": "greeting"} } }
  3. Add an element for your component inside ui/main.reel/main.html <div data-montage-id="greeting"></div>
  4. Add some content to the ui/greeting.reel/greeting.html <div data-montage-id="greeting" class="Greeting">Hello World!</div>
  5. Refresh this page

Run tests

  1. Install development dependencies npm install
  2. Run your tests
